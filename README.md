# Maths Tutor AI 🤖📝

An AI-powered math tutor designed for UK Primary School students (Ages 5-11). It generates personalized word problems based on a child's interests and uses a Spaced Repetition System (SRS) to ensure long-term mastery.

## Key Features
- **Personalized Learning**: AI tailors questions to the child's name, hobbies, and pets.
- **Adaptive Support**: Provides scaffolding hints instead of just giving answers.
- **Spaced Repetition**: Tracks mastery and schedules reviews for optimal retention.
- **Kid-Friendly Interface**: Fun, engaging, and easy to use.

## Tech Stack
- **Framework**: [Next.js](https://nextjs.org) (App Router)
- **Database**: SQLite with [Prisma](https://www.prisma.io)
- **AI**: [Google Gemini](https://ai.google.dev/)
- **Styling**: Tailwind CSS
- **State Management**: React Hooks & Server Actions
